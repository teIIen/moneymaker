# Moneymaker2 📈🤖

**Moneymaker2** — это кроссплатформенный терминал для алгоритмического трейдинга (Windows/Debian headless). Вся работа с ботом ведется через CLI и Telegram.

Проект строится на строгих принципах **SOLID**, жесткой модульности и асинхронной архитектуре (`asyncio`). На данный момент проект находится на стадии **Live Paper Trading** с поддержкой ручной валидации сделок.

## 🧠 Торговая стратегия (Math Base)
Ядром принятия решений является **VSA (Volume Spread Analysis)**. 
Математическая база основана на индикаторах **SmokeFX**, логика которых перенесена из Pine Script в Python (`pandas`, `numpy`):
- **True Volumes:** Поиск аномальных усилий и кульминаций на рынке.
- **PROfile (Liquidity Levels):** Динамический поиск пулов ликвидности (Swing High/Low) с защитой от заглядывания в будущее.
- **Kraya:** Определение экстремальной перекупленности и перепроданности.
- *В планах: Tumba, FiboChannel, MTF (Multi-Timeframe Analysis).*

## 🛠 Стек технологий
- **Язык:** Python 3.11+
- **Связь с биржами:** `ccxt` (async)
- **Обработка данных:** `pandas`, `numpy`
- **Интерфейс:** `aiohttp` (Telegram API), `logging` (Rotating Files).
- *В планах: `Textual` / `Rich` (TUI терминал).*

## 🏗 Архитектура проекта
1. **Data Ingestion (`core_lib`)** — получение OHLCV данных с биржи.
2. **Strategy Engine** — математика индикаторов и генерация DTO `TradeSignal`.
3. **Risk Management** — автоматический расчет Position Sizing на основе % риска и Stop-Loss.
4. **Execution Core** — `LivePaperTrader` для симуляции торгов в реальном времени и `TelegramListener` для связи с внешним миром.
5. **Config & Logging** — Безопасное хранение ключей и ротация `.log` файлов.

## 👥 Ролевая модель Telegram
Бот интегрирован с Telegram и поддерживает двух пользователей:
- **Архитектор/Владелец:** Получает отчеты о работе системы, запуске цикла и закрытых сделках (PNL).
- **Валидатор (Трейдер-эксперт):** Получает сигналы на вход с кнопкой "✅ Согласен с предположением". Служит дополнительным фильтром перед открытием сделки.

## 🚀 Установка и запуск
1. Клонировать репозиторий.
2. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Настроить конфигурацию:
   - Скопировать `config/settings.example.py` в `config/settings.py`.
   - Вписать свой Telegram Bot Token и Chat ID Владельца / Трейдера.
4. Запуск бэктеста:
   *Закомментируйте `asyncio.run(run_live_bot())` и раскомментируйте логику бэктеста в `main.py` (если необходимо).*
5. Запуск Live режима (Event Loop):
   ```bash
   python main.py
   ```
   *После запуска отправьте боту `/start` в Telegram, чтобы проверить пинг биржи.*
