# Moneymaker2 📈🤖

**Moneymaker2** — это кроссплатформенный (Windows/Debian headless) терминал для алгоритмического трейдинга. Вся работа с ботом ведется исключительно через CLI (Command Line Interface).

Проект строится на строгих принципах **SOLID**, жесткой модульности и асинхронной архитектуре.

## 🧠 Торговая стратегия (Math Base)
Ядром принятия решений является **VSA (Volume Spread Analysis)**. 
Математическая база основана на индикаторах **SmokeFX**, логика которых переносится из Pine Script в Python (на базе `pandas` и `numpy`):
- **True Volumes:** Поиск аномальных усилий и кульминаций на рынке.
- **PROfile (Liquidity Levels):** Динамический поиск пулов ликвидности (Swing High/Low).
- **Kraya:** Определение перекупленности/перепроданности и экстремумов (Mean Reversion).
- *В планах: Tumba, FiboChannel.*

## 🛠 Стек технологий
- **Язык:** Python 3.11+
- **Связь с биржами:** `ccxt` (async support)
- **Обработка данных (Math):** `pandas`, `numpy`
- **Интерфейс:** `Textual` / `Rich` (TUI терминал)
- **Архитектура:** `asyncio`, Decoupling.

## 🏗 Архитектура проекта
Проект разделен на независимые модули (Single Responsibility):
1. **Data Ingestion (`core_lib`)** — получение свечей и тиков с биржи через REST/WebSockets.
2. **Strategy Engine** — математика индикаторов и логика генерации сигналов.
3. **Risk Management** — жесткие правила управления капиталом (Position Sizing, Stop-Loss).
4. **Execution Core** — исполнение ордеров (Paper Trading -> Live Trading Bybit/Binance).
5. **Terminal UI** — вывод логов, позиций и графиков в консоль.

## 🚀 Установка и запуск (в разработке)
1. Клонировать репозиторий.
2. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Тестовый запуск стратегии на исторических данных:
   ```bash
   python main.py
   ```
