# Moneymaker2 📈🤖

**Moneymaker2** — это профессиональный аналитический сканер для рынков криптовалют и Forex. Бот параллельно собирает данные по десяткам торговых пар, проводит глубокий технический и фундаментальный анализ, и отправляет структурированную аналитическую сводку трейдеру в Telegram.

Проект строится на принципах модульности, SOLID и асинхронной архитектуре (`asyncio`), что позволяет ему сканировать весь рынок за секунды.

## 🧠 Аналитическое Ядро
Бот не просто ищет паттерны, он "понимает" рыночный контекст по 4 ключевым осям:

1. **Глобальный тренд (Direction):** Определение среднесрочного движения цены с помощью EMA-фильтров. Сделки открываются только по тренду.
2. **Зоны Ликвидности (Targets):** Математический поиск областей скопления покупателей и продавцов (Support/Resistance). Используется для выставления умных Stop-Loss и Take-Profit.
3. **Объемы рынка (VSA Base):** Анализ спредов и тиковых объемов для выявления силы покупателя или продавца (кульминации, аномальные усилия).
4. **Сентимент и Новости (Macro):** Сбор и NLP-анализ финансовых новостей по конкретным парам для оценки вероятности резких рывков (в разработке).

## 🛠 Стек технологий
- **Язык:** Python 3.11+
- **Источники данных:** 
  - `ccxt` (Crypto) - публичное подключение к криптобиржам без ключей.
  - `tiingo` (Forex & News) - институциональные данные и новости через REST/WebSockets.
- **Обработка данных:** `pandas`, `numpy`, `ta` (для EMA)
- **NLP (Анализ текстов):** `nltk` / `vaderSentiment` (в планах)
- **Интерфейс:** `aiohttp` / Telegram API
- **Логирование:** `logging`

## 🏗 Архитектура проекта
1. **Data Provider (`core_lib`)** — умный роутер, получающий OHLCV данные.
2. **Strategy Engine** — глубокий технический анализ (Тренд, VSA, Ликвидность) и генерация `TradeSignal`.
3. **News Analyzer** — сбор новостей и сентимент-анализ (в планах).
4. **Risk Management** — умный расчет SL и TP с привязкой к графику, а также Position Sizing.
5. **Execution Core** — `TelegramNotifier` для отправки аналитических сводок трейдеру (Market Screen).

## 🚀 Установка и запуск
1. Клонировать репозиторий.
2. Установить зависимости:
   ```bash
   pip install -r requirements.txt
   ```
3. Настроить конфигурацию (`config/settings.py`):
   - Вписать свой Telegram Bot Token и Chat ID.
   - Указать API-ключ Tiingo (потребуется для Форекса и Новостей).
   - Настроить список `TRADING_PAIRS` (до 21 пары).
4. Запуск бота-сканера:
   ```bash
   python main.py
   ```
